Console Chess
===
Gameplay
===
                Lowercase letters -- Black pieces
                Uppercase letters -- White pieces
                @ -- Black King
                # -- White King

                "Q/q" -- Queens
                "R/r" -- Rooks
                "B/b" -- Bishops
                "K/k" -- Knights
                "#/@" -- Kings

                In order to move the pieces you should
                enter moves in classical algebraic notation

Start
===
                Just run "\x64\Debug\main.exe"
